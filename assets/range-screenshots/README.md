# Range Screenshot Mapping

The Ranges page "View Source Chart" button looks up screenshots by game type and position.

## Folder layout

- `assets/range-screenshots/tournament/`
- `assets/range-screenshots/cash/`

## File naming

Use uppercase position names as the filename:

- `D.png`
- `SB.png`
- `BB.png`
- `UTG.png`
- `MP1.png`
- `MP2.png`
- `MP3.png`
- `HJ.png`
- `CO.png`

## Example

For Tournament + D position, place the screenshot at:

`assets/range-screenshots/tournament/D.png`

The modal will automatically show this image for any hand selected in that game type and position.
